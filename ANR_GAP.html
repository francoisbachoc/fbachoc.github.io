ANR project GAP

GAussian Processes for computer experiments and machine learning: more guarantees and broader applications

Summary of the project:

This project aims for new mathematical guarantees and applications for Gaussian processes. The four main themes are the following.

  • Deep Gaussian processes
  • Variational inference
  • Constrained Gaussian processes
  • New applications
